# Web Workspace

Some IELTS Buddy capabilities are browser experiences rather than MCP operations. Open or provide the relevant production route:

| Capability | URL |
| --- | --- |
| Learning center | `https://ieltsbuddy.igocn.cn/learning-center` |
| Practice center | `https://ieltsbuddy.igocn.cn/practice` |
| Mock tests | `https://ieltsbuddy.igocn.cn/mock` |
| Preparation information | `https://ieltsbuddy.igocn.cn/prep-info` |
| Courses | `https://ieltsbuddy.igocn.cn/courses` |
| Learner profile and footprints | `https://ieltsbuddy.igocn.cn/footprints` |
| Study plans | `https://ieltsbuddy.igocn.cn/plans` |
| Toolbox | `https://ieltsbuddy.igocn.cn/ai-apps` |

Toolbox experiences include reading vocabulary extraction, listening dictation, vocabulary practice, speaking topic weaving, and band-score calculation.

When the current Agent can control the IELTS Buddy web workspace, open the requested route and verify the visible state. Otherwise provide the deep link and continue any non-web coaching in the conversation.
