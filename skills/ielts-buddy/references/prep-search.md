# Preparation Search

Use these tools for current IELTS Buddy preparation content:

- `ielts_prep_search`: search published IELTS preparation guides and prep records.
- `ielts_prep_read_guide`: read one full published guide by slug.

## Workflow

1. Search with the user's topic, question, exam date, or desired subject.
2. Keep guide and record filters narrow when the user names a subject or date.
3. Read the full guide before giving a detailed summary based on it.
4. Distinguish published guidance, returned records, and your own inference.
5. Include the title, date, subject, slug, or stable identifier needed to continue.

Do not describe any preparation record as guaranteed. When search returns no matching record, say so and offer a broader query instead of fabricating a result.
